mal_error_e mal_circular_buffer_peek ( mal_circular_buffer_s *  buffer,
uint64_t  index,
void *  data 
)

This function will return the value at index in the buffer without popping the value.

Parameters
bufferThe circular buffer to peek from.
indexThe index to peek. This is from the read perspective. This means index 0 is the oldest data in the buffer and the last index is the newest data in the buffer.
dataThe returned data.
Returns
Returns MAL_ERROR_OK on success. If the buffer is empty, it will return MAL_ERROR_EMPTY.
