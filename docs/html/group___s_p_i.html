typedef mal_error_e(* mal_spi_master_transaction_complete_t) (mal_spi_msg_s *msg, mal_spi_msg_s **next_msg)

This callback will be executed at the end of a transaction where the MCU is the master.

Parameters
msgThe message of the transaction containing only the received data.
next_msgThe next message to send. Set to #NULL if no message should be transmitted.
Returns
At this point, the return code is not taken into account.
