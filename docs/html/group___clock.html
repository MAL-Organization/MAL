Will return the source clock a given timer.

Parameters
timerValid values are defined by mal_hspec_timer_e.
clockA pointer of type uint64_t. If the function returns MAL_ERROR_OK, clock will contain the frequency in hertz.
Returns
Return MAL_ERROR_OK on success.

Possible clock sources for the MCU.

Enumerator
MAL_HSPEC_SYSTEM_CLK_SRC_INTERNAL 

Source is an internal oscillator.

MAL_HSPEC_SYSTEM_CLK_SRC_EXTERNAL 

Source is an external crystal or oscillator.

MAL_HSPEC_SYSTEM_CLK_SRC_AUTO 

MAL will select the source. Note it is safer to define a known source.
