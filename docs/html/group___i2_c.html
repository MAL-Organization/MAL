This callback is used when a transaction completes.

Parameters
interfaceThe interface the transaction completed on.
packetThe packet received.
resultThe result of the transaction.
next_msgThe next message to send. This message will be sent immediately.
Returns
Return a status once you executed your callback. For now, nothing is done with this status.

enum mal_i2c_result_e

Possible I2C transactions results.

Enumerator
MAL_I2C_SUCCESS 

Successful transaction.

MAL_I2C_NACK_COMPLETE 

Transaction completed, but device did not acknowledge.

MAL_I2C_NACK_INCOMPLETE 

Transaction is incomplete, and the device did not acknowledge.

MAL_I2C_BUS_ERROR 

An error occurred on the bus during the transaction.
