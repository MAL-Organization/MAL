◆ mal_watchdog_init()

mal_error_e mal_watchdog_init ( uint32_t  timeout_ms)

This function will initialize and start the watchdog.

Parameters
timeout_msA uint32_t that contains the watchdog timeout in ms. If the watchdog is not fed before this timeout is over, the MCU will reset.
Returns
Returns MAL_ERROR_OK on success.
Note
Watchdog clock sources may have a large delta. Check your MCU's datasheet and choose a timeout accordingly.
