Detailed Description

Object pool utility. This will allow to emulate a memory heap while having a deterministic memory usage.

This function will attempt to allocate a free object.

Parameters
poolThe pool of objects to allocate from.
objectIf a free object is found, it will be stored in object.
Returns
If an object is found, MAL_ERROR_OK will be returned. MAL_ERROR_EMPTY will be returned if no free object remains.
